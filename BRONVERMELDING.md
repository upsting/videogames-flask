# Bronvermelding

Dit project gebruikt externe bronnen voor data, framework en typografie.

## Gebruikte websites

1. Data (games en categorieen)
- URL: https://my-json-server.typicode.com/upsting/videogames-api
- Gebruik in project: ophalen van gamegegevens via de endpoints `/games`, `/games/<id>` en `/categories`.

2. Flask documentatie
- URL: https://flask.palletsprojects.com/
- Gebruik in project: opzet van routes, templates en basisstructuur van de webapp.

3. Requests documentatie
- URL: https://requests.readthedocs.io/
- Gebruik in project: uitvoeren van HTTP GET-requests naar de JSON API.

4. Google Fonts
- URL: https://fonts.google.com/
- Gebruik in project: lettertypes Bebas Neue en Space Grotesk in de frontend.

## Opmerking
De code en UI zijn verder zelfstandig uitgewerkt voor deze opdracht.
